Antibiotic Resistance Predictor 🧬

📌 Problem

Antibiotic resistance is a major global health issue. This project predicts whether a bacterium is resistant or susceptible to an antibiotic using machine learning.

 📊 Dataset

We used a publicly available antibiotic resistance dataset containing bacterial isolates and their responses to antibiotics.

⚙️ Approach

Data preprocessing
 Model training using Random Forest
Prediction of resistance (Resistant / Susceptible)

📈 Results

* Model Accuracy: ~85% (approx)
* Feature importance analysis performed

🚀 How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run the notebook:
   jupyter notebook

🧠 Future Scope

 Add real-time prediction tool
 Improve accuracy with advanced models
 📊 Output
The model was trained successfully and achieved high accuracy. Feature importance visualization was used to identify key factors affecting predictions.
